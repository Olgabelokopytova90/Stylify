# Stylify

Stylify is a fashion-tech web application that helps users visualize outfits on a virtual avatar and receive AI-powered clothing recommendations.

The project focuses on combining modern frontend technologies with AI-driven logic to explore virtual try-on and personalized styling experiences.

---

## Features

- Upload and manage user photos
- Visualize clothing on a virtual avatar
- Interactive UI for trying different clothing combinations
- Modular client–server architecture

---

## 🛠 Tech Stack

### Frontend
- React
- TypeScript
- Vite
- CSS / modern styling

### Backend
- Node.js
- Express

### AI / Data
- Supabase / PostgreSQL (if used for persistence)
